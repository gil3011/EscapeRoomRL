# EscapeRoomRL

Six rooms, six reinforcement-learning algorithms, one escape-room game. Train an agent in
each room, watch it learn, then watch it escape — the faster the escape, the higher the score.

## Rooms

| # | Algorithm | Task |
|---|---|---|
| 1 | Dynamic Programming | known-model 10x10 grid with slippery cells |
| 2 | Monte Carlo control | unknown-model 10x10 grid |
| 3 | SARSA | unknown-model 10x10 grid |
| 4 | Q-Learning | unknown-model 10x10 grid |
| 5 | DQN (function approximation) | continuous (x, y, vx, vy), 10x10 m room |
| 6 | DQN (function approximation) | Room 5 + dynamic obstacles, partial observability |

See [plan.md](plan.md) for the full design and [SPRINTS.md](SPRINTS.md) for the build plan.
[PROGRESS.md](PROGRESS.md) tracks what's actually been built so far.

## Run

```bash
python -m venv .venv
.venv\Scripts\pip install -r requirements.txt
.venv\Scripts\streamlit run app.py
```

The app opens at http://localhost:8501.

## Run tests

```bash
.venv\Scripts\pytest
```

## Project structure

```
EscapeRoomRL/
├── app.py              # lobby page + room navigation
├── pages/              # one Streamlit page per room
├── engine/             # environments, agents, training runner, persistence, scoring
├── ui/                 # shared Plotly/Streamlit rendering helpers
├── docs/               # per-room theory + parameter glossary (rendered by each Info tab)
├── runs/               # gitignored — training history/checkpoints, regenerated by training
└── tests/              # unit tests for the engine modules
```
